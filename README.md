# KIVUPASS
🎫 Kivupass - Billetterie Premium RDC Kivupass est une solution de billetterie numérique moderne et sécurisée, spécifiquement conçue pour le marché de la République Démocratique du Congo. Le projet offre une expérience utilisateur fluide pour l'achat, la gestion et la validation de tickets d'événements prestigieux.
